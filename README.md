## CALVIN Network Flow Optimization Model

An open-source Python implementation of California's hydroeconomic optimization model, the California Value Integrated Network (CALVIN).

#### Getting Started

Installation and quick start instructions available [here](https://app.gitbook.com/@ucd-calvin/s/calvin/start/installation).

#### Documentation 

- [Full online documentation](https://app.gitbook.com/@ucd-calvin/s/calvin/) (raw content via the [./documentation](./documentation) directory)
- [Initial writeup on pyVIN](./documentation/references/pyvin_documentation.pdf) which refers to an earlier version of the model using Pyomo's `AbstractModel` (the setup is mostly the same in the current `ConcreteModel`). 
- Detailed [Pyomo documentation](http://www.pyomo.org/documentation/)
